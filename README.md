## 3 编译运行环境说明

### 3.1 整体说明

1. 参赛选手所提交的源码压缩包，由系统解压后自动编译运行，生成竞赛结果和回放文件；
2. 运行参赛选手源码的操作系统为 Linux 操作系统；
3. 参赛选手的源码打包脚本为 CodeCraft_zip.sh；
4. 执行机系统环境：
    操作系统版本：基于 x86 的 ubuntu 18.04
    CPU： 2 核可用 CPU
    内存： 3.8 GB 可用内存

### 3.2 编译环境

1. C/C++
    make： GNU Make 4.1
    cmake： cmake version 3.22.0
    gcc： gcc 7.3.0
    g++： g++ 7.3.0
